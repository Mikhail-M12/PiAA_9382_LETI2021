"#" в именах файлов "task#.exe" обозначает номер задания на Stepik.
Индивидуализация работает для обоих заданий.

Примеры входных данных из директории input_examples\ вида "task#_*.txt" предназначены для задания #, * - номер теста.

В директории src\ лежат файлы с кодом:
1) без возможности вывода промежуточных данных ("alg_lab5_clear.cpp")
2) финальная версия ("alg_lab5.cpp"), вывод промежуточных данных можно отключить, закомментировав строку "#define DEBUG"

Также в коде можно выбрать задание, установив соответствующий ему номер * в строку "#define TASK *"
