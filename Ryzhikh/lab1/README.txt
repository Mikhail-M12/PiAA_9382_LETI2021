Первоначально надо создать exe файл с помощью команды "make" в директории проекта. 
Для проверки прописать "auto.sh", скрипт возьмет файлы из папки 
test и прогонит их через программу, результат запишет в папку results.

Отладочную информацию можно убрать, раскомментив define.
Отладочной информации очень много из-за рекурсивной работы алгоритма и
из-за принципа его работы: методом подбора минимального количества квадратов.
